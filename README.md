## Building

You'll need [Node.js](https://nodejs.org) installed on your computer in order to build this app.

```bash
git clone https://github.com/andrehidalgo93/DIO-react-native-video-capture.git
$ cd DIO-react-native-video-capture
$ npm install
$ npm run start
```
